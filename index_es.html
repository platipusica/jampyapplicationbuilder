Construye aplicaciones empresariales Jam-py
Qué es Jam.py? Lo que MS Access es para equipos de escritorio, Jam.py es para Web. Y mucho más. Sencillo.

Con Jam.py puedes crear, personalizar, probar y compartir aplicaciones impresionantes, rápidas y orientadas a eventos para SQLite, Oracle, MySQL, PostgreSQL y Firebird. Todo eso para gratis y no hay bloqueo de proveedores!

De una idea de vivir sitio web en poco tiempo.

Desarrolla

Marco orientado a objetos, basado en eventos con estructura jerárquica.

Personaliza

Tratamiento de datos de tabla de base de datos potente pero fácil en el servidor y el cliente.

Entrega

Forma segura de administrar cuentas de usuario y contraseñas.

Principio DRY! No te repitas, hazlo una vez, hazlo bien.

Con la aplicación Jam.py Bases de datos Builder, puede resolver un problema comercial específico. Fuera de la caja Jam es proporcionando: acceso rápido a bases de datos subyacentes, seguridad, autenticación, validación, calendarios, multi idiomas, todo eso con   mínimo de codificación necesaria. Siendo un framework Python, es extensible y flexible.

Construir aplicaciones de base de datos muy rápido, de forma gratuita.

Con Jam.py, sólo necesitas el navegador! Su productividad será el cohete de cielo a medida que cavar en el marco utilizando nuestro editor de código. Realice cambios en el editor, actualice la página y vea Los resultados.

  • Entorno de desarrollo basado en web.
  • Editor de código con resaltado de sintaxis y sugerencias de código.
  • Sin opciones declarativas, usted está a cargo.
  • Instante WYSIWYG
  • Seguimiento del ciclo de vida de la aplicación.
  • SQL y procedimientos almacenados soportados por los principales proveedores.
  • Integre cualquier biblioteca de Python sin bloqueo de contrato y reduzca el coste al instante.
  • Bootstrap, JQuery, JS, todo aquí. Úsalo sin problemas y aprendiendo estas masivas bibliotecas.
  • Cree informes ricos e informativos con las plantillas de LiberOffice.
  • Dispositivos móviles soportados fuera de la caja.

Redes de eventos

Las redes activadas por eventos le permiten manipular fácilmente datos simplemente haciendo clic en una celda y editando su valor.

  • Los controles de interfaz visual de datos orientados a eventos hacen que el marco sea flexible y potente.
  • Edite sus datos en la cuadrícula, como lo haría en cualquier aplicación de hoja de cálculo de escritorio.
  • Cree la tabla de detalle maestro con brisa, utilizando plantillas para mostrar, que no es más que una copia / pega. Fácil. De nuevo, no hay métodos declarativos, el control es con usted como debe ser.

jsCharts o cualquier biblioteca de gráficos

Bloqueado con una capacidad de gráficos de proveedores? Nunca más. Utilizar bibliotecas gratuitas como jsChart, y el resto.

  • Utilice las mismas capacidades de gráficos en sus dispositivos móviles, de una vez por todas.
  • Visualice gráficos inmediatamente después de crear / importar tablas, con unas pocas líneas de código. Simple y eficaz.
  • Analizando/mostrando datos GRANDES? Añadir libre Python lib, construir un formulario Web Jam con los parámetros, ejecutar en el servidor. Lucro.
Simple y eficaz interfaz de usuario sin ningún problema

Diseñado con la simplicidad en la mente, basado en la fundación fuerte de Delphi. No hay widgets web, no hay dependencias de tales. Sin embargo, puede utilizar CSS o JS y lograr resultados con Jam. En organizaciones pequeñas y medianas sin desarrolladores, uno puede construir una aplicación basada en requisitos en ningún momento con sin prisas de marcos declarativos.

La implementación de una aplicación Jam.py tarda unos segundos. Literalmente.

Sólo hay unos pocos archivos que desplegar en el entorno de desarrollo / producción. Comparando con otros con mega despliegue o dependencias de bases de datos, Jam es un ligero como una pluma.

  • Rápida exportación / importación de aplicaciones, fácilmente transportable, kilobytes de tamaño.
  • Desarrollar en una base de datos, desplegar en el otro. Esta es nuestra promesa a usted. No hay bloqueo de base de datos. Nunca.
  • Hacer una aplicación más rápida es extremadamente fácil. Desarrollar una vez, desplegar en cualquier lugar.
  • Jam.py se desplegará en Windows, así como en Mac o Linux. ¿Pi de la frambuesa? Sin preocupaciones.
